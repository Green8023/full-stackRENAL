# full-stackRENAL
AI empowers full-stack smart diagnosis

This project developed a full-stack diagnostic model for renal cell carcinoma, including four tasks: tissue detection, subtype prediction, auto-grading and prognostication. It is completed based on the features extracted by Prov-GigaPath combined with the abmil architecture.
